# Javascript Todo App 📆

> 달력의 날짜를 선택후, 해당 날짜의 Todo List를 추가, 수정, 삭제 하는 프로젝트

## Skills

- html
- css
- Vanilla JS

## Functions

- new Date를 이용한 달력 커스텀
- Local Storage로 Todo Item 관리
- 더보기 버튼 drop down 커스텀
